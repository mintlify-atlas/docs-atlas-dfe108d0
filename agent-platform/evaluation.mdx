Evaluation Studio is a unified workspace for evaluating AI system performance across two areas: **Model Evaluation** and **Agentic Evaluation**.

```
Define Criteria → Load Test Data → Run Evaluations → Analyze Results → Iterate
```

## Evaluation types

<CardGroup cols={2}>
  <Card title="Model evaluation" icon="chart-bar" href="/agent-platform/evaluation/model-evaluation">
    Assess LLM performance using configurable quality and safety metrics. Upload datasets, apply built-in or custom evaluators, and analyze results through visual scoring.
  </Card>
  <Card title="Agentic evaluation" icon="robot" href="/agent-platform/evaluation/agentic-evaluation">
    Assess how effectively an agentic app performs in production and pre-production environments. Import real sessions or generate simulated sessions using personas and test scenarios.
  </Card>
</CardGroup>

### Model evaluation

Model evaluation enables you to:

- Upload datasets with input-output pairs, or generate outputs from a deployed model.
- Apply Quality, Safety, and RAGAS evaluators—or create custom evaluators.
- Integrate external outputs via Run a Prompt, Run an API, or Run Search AI.
- Analyze model effectiveness through visual scoring, pass/fail thresholds, and exportable results.

**Use when**: Fine-tuning, comparing, or validating models before or after deployment.

### Agentic evaluation

Agentic evaluation enables you to:

- Import app sessions and traces from production or simulations.
- Generate simulated sessions using Personas and Test Scenarios.
- Run multi-level evaluations across sessions and traces.
- Analyze inputs and outputs across supervisors, agents, and tools.

**Use when**: Pre-deployment validation, debugging agentic workflows, or ongoing production quality monitoring.

## Accessing Evaluation Studio

1. Log in to your platform account.
2. From the modules menu, select **Evaluation Studio**.
3. On the Evaluation page, select **Model Evaluation** or **Agentic Evaluation** to begin.
