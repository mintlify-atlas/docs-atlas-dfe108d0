### Custom model integration

To use a custom third-party model in agentic apps, it must meet two requirements:

**Tool calling support (mandatory)**: The model must natively support tool calling. Enable it explicitly during model configuration.

**Compatible API structure**: The request and response structure must follow either:
- **Anthropic** – Messages API
- **OpenAI** – Chat Completions API

### Structured output support

Platform-hosted models can produce structured JSON responses. Requires no optimization or vLLM only.

<Note>CT2-optimized, fine-tuned, Hugging Face imported, and locally imported models do not support structured output.</Note>
