The following Python libraries are pre-imported and available in the Function Node script editor.

- [json](#json)
- [enum](#enum)
- **Numerical libraries**: [NumPy](#numpy), [pandas](#pandas), [collections](#collections), [math](#math), [cmath](#cmath), [statistics](#statistics), and [random](#random).
- **String manipulation libraries**: [re](#re) and [textwrap](#textwrap).
- **Date-time libraries**: [datetime](#datetime), [time](#time), and [calendar](#calendar).
- [Itertools](#itertools)

Libraries are imported in the Function Node as follows:

```python
import json
import numpy
import scipy
import pandas
import collections
import math
import cmath
import statistics
import random
import enum
import re
import textwrap
import datetime
import time
import calendar
import itertools
```

## json

Functions: `JSONDecodeError`, `JSONDecoder`, `JSONEncoder`, `dumps`, `loads`.

## enum

Functions: `DynamicClassAttribute`, `Enum`, `Flag`, `IntEnum`, `IntFlag`.
