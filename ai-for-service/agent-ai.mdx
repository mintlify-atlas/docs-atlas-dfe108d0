Agent AI provides real-time AI assistance to human agents during customer interactions, boosting productivity and improving customer satisfaction.

## How it works

Agent AI monitors live conversations and surfaces contextual suggestions, knowledge results, and recommended actions in a widget embedded in the agent's console.

1. Customer sends a message or speaks during a voice call.
2. Agent AI analyzes the conversation context in real time.
3. The widget surfaces suggested responses, relevant knowledge articles, and next-best actions.
4. The agent accepts, modifies, or ignores the suggestions.
5. Feedback from agent edits improves suggestion quality over time.

## Features

| Feature | Description |
|---------|-------------|
| **Suggested responses** | Contextual response suggestions appear as the customer speaks. |
| **Automated summaries** | Real-time and after-call conversation summaries. |
| **Knowledge assistance** | Integrated knowledge search triggered automatically by context. |
| **Next-best-action** | Recommended actions based on context, sentiment, and customer tier. |
| **Real-time coaching** | On-screen hints and nudges triggered by speech patterns. |
| **Playbooks** | Step-by-step guidance for handling specific interaction types. |
| **Agentic Copilot** | Autonomous conversation handling with agent supervision controls. |

## Configuration

### Enable Agent AI

1. Navigate to **Agent AI** > **Configuration**.
2. Enable Agent AI for desired queues.
3. Configure feature settings.
4. Test with pilot agents.

### Feature settings

| Setting | Options |
|---------|---------|
| **Suggestion mode** | Continuous, On-demand, Disabled. |
| **Auto-summary** | Enabled, Disabled. |
| **Knowledge search** | Auto, Manual, Both. |
| **NBA** | Enabled, Disabled. |

### Integration

Connect Agent AI to:

- **Search AI** — for knowledge retrieval.
- **CRM** — for customer context.
- **Case management** — for action execution.
- **Quality AI** — for coaching feedback.

## Agent experience

Agent AI appears as a widget in the agent console with five tabs: **Search**, **My AI Agent (Bot)**, **Assist**, **Transcript** (voice only), and **More**.

[Learn more about the agent experience →](/ai-for-service/agentai/agent-experience)

## Analytics

| Metric | Description |
|--------|-------------|
| **Suggestion acceptance** | Percentage of suggestions used. |
| **Time saved** | Handle time reduction. |
| **Knowledge utilization** | Searches and clicks. |
| **NBA conversion** | Recommended actions taken. |

## Best practices

- Start with a pilot group of agents.
- Gather feedback and iterate before full rollout.
- Keep the knowledge base current and remove outdated content.
- Introduce Agent AI in agent training to explain the feedback mechanism.
