## Overview dashboards

Access overview dashboards under **Analytics** > **Overview**.

| Dashboard | Description |
|-----------|-------------|
| **Summary Dashboard** | Snapshot of AI Agent conversations, user analytics, and NLP/execution performance |
| **Conversations Dashboard** | Insights into conversation handling including self-service rates, agent transfers, and drop-offs |
| **Conversations History** | Review past conversation transcripts with prebuilt and custom filters |
| **Users Dashboard** | Actionable insights on unique, new, and returning users with trend analysis |
| **Feedback Dashboard** | Analytics on NPS, CSAT, and Like/Dislike survey responses |
| **Conversations** | Detailed list of AI Agent and agent-handled conversations with search, filtering, and export |

## GenAI analytics

Access GenAI analytics under **Analytics** > **GenAI Analytics**.

| Dashboard | Description |
|-----------|-------------|
| **GenAI Dashboard** | Overview of LLM token usage and performance — total requests, success rate, and latency |
| **Performance Analytics** | Detailed LLM performance table by model and feature combination |
| **Gen AI Logs** | Request-level details for all LLM calls including token counts and guardrail outcomes |

## Automation AI analytics

Access Automation AI analytics under **Analytics** > **Automation AI**.

| Dashboard | Description |
|-----------|-------------|
| **Performance Dashboard** | Post-publication NLP and integration health metrics |
| **Conversation Insights** | Clusters user utterances into semantic groups to support utterance training |
| **NLP Insights** | Analytics on intent identification and unhandled utterances |
| **Task Execution Logs** | Execution-level analytics for tasks, APIs, scripts, and debug logs |
| **Custom Dashboard** | Build business-specific dashboards using Analytics, Messages, and Sessions datasets |
| **Conversation Flows** | Visual map of user journeys showing popular intents and drop-off points |

## Search AI analytics

Access Search AI analytics under **Analytics** > **Search AI**.

| Dashboard | Description |
|-----------|-------------|
| **Answer Insights** | Analytics for each query-answer interaction, including answer quality and user feedback |

## Contact Center AI analytics

Access Contact Center analytics under **Analytics** > **Contact Center AI**.

| Dashboard | Description |
|-----------|-------------|
| **Contact Center Metrics** | Reference listing of all available metrics in Contact Center AI |
| **Agent AI Dashboard** | Performance metrics for Agent AI suggestions, relevance, and automation outcomes |
| **Agent AI Conversation Logs** | Library of all agent-customer conversations with AI-generated summaries |
| **Queues & Agents** | Real-time and historical operational metrics across queues and agents |
| **Reports** | Scheduled and on-demand performance reports with PDF/CSV export and email delivery |
| **Wallboards** | Real-time KPI display for supervisors in Center Wide and Queue Specific views |

## Quality AI analytics

Access Quality AI analytics under **Analytics** > **Quality AI**.

| Dashboard | Description |
|-----------|-------------|
| **Scheduled Reports** | Monitor agent performance and quality adherence with on-demand or scheduled report delivery |
| **QM Reports List** | Three pre-defined report types: Agent Performance, Interaction Evaluation & Conversation Analytics, and Evaluation Form Summary |

## Exporting analytics data

Most dashboards support data export:
- **CSV** — Download raw data for further analysis in spreadsheet tools
- **PDF** — Generate formatted reports for sharing with stakeholders

To export data, click the **Export** or **Download** button in the top-right corner of any dashboard.

## Date range and filters

All analytics dashboards support filtering by:
- **Date range** — Select preset ranges (today, last 7 days, last 30 days) or custom date ranges
- **Channel** — Filter by voice, web chat, messaging, or all channels
- **Bot/Assistant** — Filter to a specific AI assistant
- **Language** — Filter conversations by language
