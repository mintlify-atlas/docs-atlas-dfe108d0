Traditional NLP methods are no longer the recommended approach for building conversational experiences. DialogGPT and the Agent Node are now the standard for agentic automation on the platform.

<Note>
  If you have existing NLP-based apps, you can continue to maintain them. Migrate to DialogGPT to leverage AI Agents and agentic capabilities. See the Upgrade Guide for migration path details.
</Note>
