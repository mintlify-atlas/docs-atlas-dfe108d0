Contact Center AI is an AI-native Contact Center as a Service (CCaaS) platform that improves customer experience, boosts revenue, and lowers operational costs. It delivers consistent, personalized experiences for both customers and agents across channels. Use Automation AI and Answers to handle routine queries automatically and escalate to a human agent when needed.

## Components

| Component | Description |
|-----------|-------------|
| [Agent Console](/ai-for-service/contact-center/agent-console) | Unified agent desktop with AI tools. |
| [Routing, Queues, and Skills](/ai-for-service/contact-center/routing-queues-skills) | Intelligent skill-based routing and queue management. |
| [Supervisor Tools](/ai-for-service/contact-center/supervisor) | Real-time monitoring and management. |
| [Campaigns](/ai-for-service/contact-center/campaigns) | Outbound voice, SMS, and proactive web campaigns. |
| [Configurations](/ai-for-service/contact-center/configurations/configurations) | Hours of operation, languages, response templates, and more. |
| [Performance Management](/ai-for-service/contact-center/performance-management/slas-and-alerts) | SLAs, alerts, and service level monitoring. |
| [Flows and Routing](/ai-for-service/contact-center/flows-and-routing/conditional-and-waiting-flows) | Conditional flows and waiting experiences. |

## Key capabilities

| Capability | Description |
|------------|-------------|
| **Omnichannel** | Voice, chat, email, SMS, and social media in one platform. |
| **Intelligent routing** | Skill-based, priority, and load-balanced routing. |
| **Agent assist** | Real-time AI suggestions via [Agent AI](/ai-for-service/agent-ai). |
| **Quality management** | Automated QA via [Quality AI](/ai-for-service/quality). |
| **Self-service** | Virtual assistants via [Automation AI](/ai-for-service/automationai). |
| **Analytics** | Real-time dashboards and historical reporting. |

## Quick start

<Steps>
  <Step title="Configure queues">
    Set up queues for different inquiry types with hours of operation, routing methods, and ACW settings.

    [Learn more about queues →](/ai-for-service/contact-center/routing-queues-skills#queues)
  </Step>
  <Step title="Define skills">
    Create skill groups and skills to categorize agent capabilities and proficiency levels.

    [Learn more about skills →](/ai-for-service/contact-center/routing-queues-skills#skills)
  </Step>
  <Step title="Configure routing">
    Set up routing rules to match customers with the right agents using skill, language, and priority matching.

    [Learn more about routing →](/ai-for-service/contact-center/routing-queues-skills#how-routing-works)
  </Step>
  <Step title="Set up the agent console">
    Configure the agent desktop layout, widgets, and status management.

    [Learn more about the agent console →](/ai-for-service/contact-center/agent-console)
  </Step>
</Steps>
