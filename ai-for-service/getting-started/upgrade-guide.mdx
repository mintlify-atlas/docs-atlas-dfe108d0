AI for Service (XO v11) is a unified platform that brings together Automation AI, Contact Center AI, Search AI, and Agent AI into a single, integrated experience. Upgrading from XO v10 unlocks advanced capabilities including DialogGPT-powered intent handling, Agentic RAG for search, and a redesigned Dialog Builder—while preserving your existing bot configurations and data.

## Key differences between v10 and v11

| Aspect | XO v10 | AI for Service (XO v11) |
|--------|--------|-------------------------|
| **Intent detection** | ML utterance training required | DialogGPT: descriptions + embeddings only |
| **Orchestration** | Rule-based dialog manager | Agentic orchestration via LLMs |
| **Search** | SearchAssist (separate product) | Search AI integrated into the platform |
| **Agent AI** | AgentAssist (separate product) | Agent AI integrated into the platform |
| **Multi-product** | Separate logins per product | Single unified platform |

## Migration path overview

<Steps>
  <Step title="Review benefits and differences">
    Understand what changes between v10 and v11 before beginning migration.
