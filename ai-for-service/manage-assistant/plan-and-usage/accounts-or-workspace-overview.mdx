title: "Account or workspace overview"
description: "Understand workspace types, roles, and enterprise upgrade options for AI for Service."
---

The platform organizes users and resources through **Workspaces**. A workspace is a container for users, applications, and access controls. Workspaces come in two types — **Standard** and **Enterprise** — each with different features, usage limits, and licensing options.

## Roles

| Role | Description |
|------|-------------|
| **Workspace Owner** | The user who created the workspace. Has access to the Administration Console and Manage Workspace. Can invite users, manage roles, and view plan and usage details. |
| **App Owner** | The user who created an app. Can invite collaborators, transfer ownership, and own multiple apps. |
| **Collaborator** | A user invited to collaborate on an app. Has limited access based on their assigned role. |

## Workspace types

### Standard workspace

All new workspaces start as Standard.

- New apps include a free trial period. After the trial, you can subscribe to the **Essential Plan** or **Advanced Plan**.
- Workspace or App Owners can request an upgrade to Enterprise.

### Enterprise workspace

Enterprise workspaces offer premium features, higher usage limits, and higher rate limits.

- Features, usage limits, and rate limits are governed by your license agreement.
- All apps in an Enterprise Workspace are assigned to the **Enterprise Usage Plan**.

## Enterprise upgrade flow

### Product-specific upgrades

Enterprise upgrades apply only to the products included in your license. For example, purchasing only Search AI upgrades the workspace for Search AI only — other products remain unchanged.

### Post-upgrade access

After the upgrade, users can immediately access licensed products.

| Before upgrade | After upgrade |
|----------------|---------------|
| **Try Now** button (Automation AI, Contact Center AI). | **Create New** button. |
| **Contact Us** button (Search AI, Agent AI). | **Create New** button. |

### Partial product upgrades

When only certain products are upgraded, users can create apps only for those products. Products without an enterprise license remain unavailable.

## Workspace linked to multiple apps

A Workspace Owner can manage Plan and Usage across multiple apps from a single view. Available actions:

- Manage free and paid credits for individual apps.
- View the number of paid and inactive apps.
- View free credit statistics.
- Upgrade the current plan or add funds.
- View **Current Credits**, **Allowed Requests**, and **Usage Trends**.
- View **Billing Request Trends** by week, month, or custom date range.
- Access **Manage Tickets** and **Community** pages.
