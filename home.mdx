## Platform overview

Kore.ai provides an integrated suite of enterprise AI products built on a shared Agent Platform foundation. Each solution addresses a distinct business domain while sharing common infrastructure for models, prompts, evaluation, and governance.

| Product | Get started | Key features |
| :--- | :--- | :--- |
| Agent Platform | [Get started](/agent-platform/getting-started) | Orchestration, Agents, Tools, Prompts, Evaluation, Guardrails |
| AI for Work | [Get started](/ai-for-work/getting-started) | Custom Agents, Pre-built Solutions, Analytics, Administration |
| AI for Service | [Get started](/ai-for-service/getting-started) | Automation AI, Search AI, Contact Center AI, Agent AI, Quality AI |
| AI for Process | [Get started](/ai-for-process/getting-started) | Workflows, Nodes, Model Hub, Monitoring |
